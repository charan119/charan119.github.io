Dense Subgraph Algorithm Performance Analysis

Introduction

Comparing performance of Exact (Algorithm 1) and CoreExact (Algorithm 4) algorithms on three datasets:

  • AS733 - Autonomous systems network
  • Netscience - Collaboration network
  • Cal-lepTh - Citation network

Analysis 1: Execution Time Growth

CoreExact shows significantly better scalability as clique size (h) increases. While both algorithms' execution times grow exponentially with h, CoreExact's growth rate is substantially slower due to its core-based optimizations.

Analysis 2: Speedup Factors

Speedup factors (Exact Time/CoreExact Time) increase with larger h values, particularly noticeable in Cal-lepTh dataset. This demonstrates CoreExact's efficiency in handling complex clique patterns through its core decomposition approach.

Analysis 3: Dataset Characteristics

Netscience shows best CoreExact performance due to its clustered collaboration structure. Cal-lepTh's citation patterns make it most challenging, but CoreExact maintains strong performance through effective core pruning.
